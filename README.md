## 🎯 解决什么问题

A 股散户炒 ETF，最头疼三件事：

1. **缠论太难学** —— 理论复杂，笔/线段/中枢/背驰手工画图费时费力，还容易画错
2. **基本面靠感觉** —— 行业景气度、政策导向、估值分位，没有系统化的研判框架
3. **数据分散** —— K线、主力资金、公告、研报分散在各个平台，拼凑分析效率极低

**ChanTheory ETF 基于东方财富妙想技能包，接入权威数据，让散户也能用机构级工具分析 ETF。**

## 📊 功能特性

### 📐 缠论结构分析
- 自动识别笔、线段、中枢、背驰
- 标注一买/二买/三买，一卖/二卖/三卖
- 支持日线、60分钟、30分钟、周线四周期切换
- 关键支撑位 / 压力位标注
- 走势分类与应对预案
- 📥 分析报告一键导出（打印PDF）
- 🔔 买卖点预警规则设置

### 🌡️ 行业景气度打分
- **五维度量化模型**：供需格局、政策导向、业绩表现、估值分位、资金流向
- SVG 雷达图可视化
- 12期景气度趋势图
- 拐点预判 + 持续性评级

### 📊 网格策略
- 基于波动率的自适应网格参数
- 止损/止盈建议

### 💰 资金研判
- 主力资金流向分析、DDX 指标追踪

### 🔍 AI 智能选股（妙想技能）
- 四维因子：资金面 × 技术面 × 基本面 × 景气度
- 预设策略：资金龙头 / 技术突破 / 低估值高息 / 景气赛道
- 自定义筛选 + 行业赛道过滤
- 结果页 TOP10，点击跳转完整分析报告

### 💼 持仓管理（妙想技能）
- 实时持仓市值、盈亏核算
- 账户资产全景展示

### 🔔 智能预警
- Cron 定时自动检查买卖点
- PushPlus / Server酱 推送通知

### 🎨 三套专业主题
- 简约白 / 科技黑 / 商务蓝，随时切换

## ⚡ 快速启动

### 1. 克隆项目

```bash
git clone https://github.com/evoo-Proxy/chan-theory-h5.git
cd chan-theory-h5
```

### 2. 安装依赖

```bash
cd client && npm install
cd ../server && npm install
```

### 3. 配置东方财富 API Key

> 🔑 **获取 API Key**：[东方财富妙想开放平台](https://dl.dfcfs.com/m/itc4)（每日 300 次免费额度）

```bash
# Linux/macOS
export MX_APIKEY=your_api_key_here

# Windows (PowerShell)
$env:MX_APIKEY="your_api_key_here"
```

### 4. 启动服务

```bash
# 终端 1：后端 API（端口 3099）
cd server && npm start

# 终端 2：前端（端口 5173）
cd client && npm run dev
```

访问 **http://localhost:5173**
